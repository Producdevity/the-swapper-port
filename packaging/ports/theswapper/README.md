## Notes

Thanks to Facepalm Games for creating The Swapper.

## Installation

Copy the Windows Steam game files into `ports/theswapper/gamedata`.

The folder should contain:

```text
TheSwapper.exe
mainSettings.ini
data/
```

On first launch, the PortMaster setup screen validates the files and prepares the local configuration. The retail game assets are not modified.

## Controls

The game has controller support. If a device does not map correctly, update PortMaster first so the current SDL controller database is available.
