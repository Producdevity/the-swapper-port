## Notes

Thanks to Facepalm Games for creating The Swapper.

## Installation

Install the port with PortMaster, then copy the Windows Steam game files into `ports/theswapper/gamedata`.

The folder should contain:

```text
TheSwapper.exe
mainSettings.ini
data/
```

On first launch, the PortMaster setup screen validates the files and prepares the local configuration. The retail game assets are not modified.

## Controls

Select Xbox/Deck Controller when prompted.

| Button             | Action                 |
| ------------------ | ---------------------- |
| Left Stick / D-Pad | Move / menu navigation |
| Right Stick        | Move cursor            |
| A                  | Jump / interact        |
| B                  | Map / PDA              |
| X                  | interact               |
| L1 / L2            | Clone                  |
| R1 / R2            | Swap                   |
| Start              | Pause / open menu      |
